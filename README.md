# Amazon Clone (HTML & CSS)
This project is a static Amazon homepage clone created using **HTML and CSS only**.  
The purpose of this project is to practice frontend layout, styling, and responsive design concepts.
## Features
- Amazon-like homepage layout
- Navigation bar with search section
- Product sections and cards
- Footer similar to Amazon
- Clean and structured UI
## Technologies Used
- HTML5
- CSS3
## What I Learned
- Building large layouts using HTML
- Styling complex pages with CSS
- Flexbox and basic responsiveness
- Organizing files in a real project
